Place black-and-white photo files in this directory and reference them from `content/photos/*.json`.

Example:

```json
{
  "title": "Rain Walk",
  "src": "/photos/street/rain-walk.jpg",
  "alt": "A person walking through rain on a city street",
  "location": "Bengaluru",
  "year": "2026",
  "aspect": "4 / 5"
}
```
